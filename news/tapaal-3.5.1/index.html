TAPAAL 3.5.1

Published 2019-04-23 - (1 min reading time)

We are glad to announce the release of TAPAAL 3.5.1.

TAPAAL 3.5.1 is a release that fixes numerous bugs in the GUI introduced in the 3.5.0 release and it also fixes trace generation issue in the untimed verifypn engine.
